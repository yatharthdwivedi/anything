Creating a git repository

  1. First install git in your device
  2. Create a id in github
  3. Make a repository in github
  4. Clone the repository in git bash / git terminal
  5. Create a branch in github
    1. Make some changes in branches as per your wish
  6. Add the changes by using certain commands
  7. Commit them
  8. After commiting the changes push them to see changes in github and your website too
